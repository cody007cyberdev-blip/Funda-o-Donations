# 🤝 Fundação Donations
Plataforma dedicada a facilitar a gestão de doações e a transparência em campanhas de caridade. Conecta doadores a causas importantes de forma simples e segura.

## ✨ Funcionalidades
- **Gestão de Campanhas:** Criação e acompanhamento de metas de doação.
- **Painel de Doador:** Histórico de contribuições e impacto gerado.
- **Transparência:** Relatórios em tempo real sobre a utilização dos fundos.

## 🛠️ Tecnologias
- **Frontend:** React + Vite
- **Gerenciador de Pacotes:** Yarn
